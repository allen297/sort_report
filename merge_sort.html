📚 介紹

合併排序法（Merge Sort）是一種高效的排序演算法，它採用了經典的「分而治之」（Divide and Conquer）策略。 它會將未排序的列表不斷地對半分割，直到每個子列表都只包含一個元素，然後再將這些子列表兩兩合併，最終組合成一個完整且排序好的列表。

🔍 基本原理

✨ 核心思想：分而治之

  1. 分割 (Divide)： 將列表從中間切半，分成左右兩個子列表，直到每個列表只剩下 1 個元素。
  2. 解決 (Conquer)： 長度為 1 的列表被視為已排序完成。
  3. 合併 (Merge)： 將兩個已排序的子列表，依大小順序合併成一個新的排序列表。

偽代碼

procedure mergeSort(A) if length(A) <= 1 then return A mid = length(A) / 2 left = A[0...mid-1] right = A[mid...length(A)-1] left = mergeSort(left) right = mergeSort(right) return merge(left, right) end procedure

💻 代碼實現

function mergeSort(arr) { if (arr.length <= 1) return arr; const mid = Math.floor(arr.length / 2); const left = mergeSort(arr.slice(0, mid)); const right = mergeSort(arr.slice(mid)); return merge(left, right); } function merge(left, right) { let result = [], i = 0, j = 0; while (i < left.length && j < right.length) { if (left[i] <= right[j]) { result.push(left[i]); i++; } else { result.push(right[j]); j++; } } return result.concat(left.slice(i)).concat(right.slice(j)); }
