📚 介紹

插入排序法是一種直觀且高效的排序演演算法，類似於整理撲克牌的方式。 它逐個取出未排序的元素，將其插入到已排序部分的正確位置。 插入排序是穩定的，且對於小規模數據或接近排序的數據效率很高。

🔍 基本原理

✨ 核心思想

  1. 將列表分為已排序和未排序兩部分
  2. 從未排序部分取出第一個元素
  3. 在已排序部分中找到正確位置
  4. 將元素插入該位置，其他元素右移
  5. 重複上述過程，直到所有元素都被插入

視覺化示例

初始列表：[5, 3, 8, 4, 2] 已排序部分：[5]，未排序部分：[3, 8, 4, 2] 第一次插入： 取出 3：[5] 中的 5 > 3，3 插入到 5 前面 結果：[3, 5] | [8, 4, 2] 第二次插入： 取出 8：[3, 5] 都 < 8，8 插入到末尾 結果：[3, 5, 8] | [4, 2] 第三次插入： 取出 4：5 > 4 但 3 < 4，4 插入到 3 和 5 之間 結果：[3, 4, 5, 8] | [2] 第四次插入： 取出 2：所有元素都 > 2，2 插入到最前面 結果：[2, 3, 4, 5, 8] ✓ 最終排序結果：[2, 3, 4, 5, 8]

📖 演演算法詳解

偽代碼

procedure insertionSort(A : list) for i := 1 to length(A) - 1 do key := A[i] j := i - 1 while j >= 0 and A[j] > key do A[j + 1] := A[j] j := j - 1 end while A[j + 1] := key end for end procedure

逐步執行

1. 初始化： 從第二個元素開始（i=1）
2. 取出元素： 將 A[i] 保存為 key（待插入元素）
3. 設定指針： j 指向已排序部分的最後一個元素 (j = i-1)
4. 比較移動： 比較 A[j] 和 key，如果 A[j] > key，將 A[j] 右移到 A[j+1]
5. 向左掃描： j 向左移動，繼續比較和移動
6. 插入元素： 找到位置後，插入 key 到 A[j+1]
7. 重複： 回到第 2 步，處理下一個元素
8. 終止： 所有元素都被插入時，排序完成

📊 複雜度分析

時間複雜度

情況 複雜度 說明
最好情況 O(n) 列表已排序，每個元素只需比較一次
平均情況 O(n²) 列表隨機排列
最壞情況 O(n²) 列表逆序排列，每個元素都需要移到最前面

空間複雜度

指標 複雜度 說明
空間複雜度 O(1) 只使用常數個變數，原地排序

比較次數

對於長度為 n 的列表，平均比較次數為：

最好情況：n-1 次比較 平均情況：n(n-1)/4 次比較 最壞情況：(n-1) + (n-2) + ... + 1 = n(n-1)/2 ≈ O(n²) 次比較

💻 代碼實現

基礎版本

def insertion_sort(arr): for i in range(1, len(arr)): key = arr[i] j = i - 1 while j >= 0 and arr[j] > key: arr[j + 1] = arr[j] j -= 1 arr[j + 1] = key return arr

二分插入排序（優化版本）

def binary_insertion_sort(arr): for i in range(1, len(arr)): key = arr[i] # 二分查找正確位置 left, right = 0, i - 1 while left <= right: mid = (left + right) // 2 if arr[mid] > key: right = mid - 1 else: left = mid + 1 # 移動元素 for j in range(i - 1, left - 1, -1): arr[j + 1] = arr[j] arr[left] = key return arr

function insertionSort(arr) { for (let i = 1; i < arr.length; i++) { const key = arr[i]; let j = i - 1; while (j >= 0 && arr[j] > key) { arr[j + 1] = arr[j]; j--; } arr[j + 1] = key; } return arr; }

function binaryInsertionSort(arr) { for (let i = 1; i < arr.length; i++) { const key = arr[i]; let left = 0, right = i - 1; // 二分查找正確位置 while (left <= right) { const mid = Math.floor((left + right) / 2); if (arr[mid] > key) { right = mid - 1; } else { left = mid + 1; } } // 移動元素 for (let j = i - 1; j >= left; j--) { arr[j + 1] = arr[j]; } arr[left] = key; } return arr; }

⚖️ 優缺點

✅ 優點

  • 易於理解和實現： 算法直觀，類似於實際排序撲克牌
  • 適應性好： 對於接近排序的數據特別高效
  • 內存高效： 原地排序，空間複雜度為 O(1)
  • 穩定排序： 保持相等元素的相對順序
  • 在線排序： 可以邊接收數據邊排序
  • 小規模數據高效： 對於小型數組比快速排序更快

❌ 缺點

  • 大規模數據低效： 時間複雜度為 O(n²)
  • 較多移動操作： 需要不斷移動元素
  • 效率不如分治算法： 快速排序、合併排序等更優

students = [ {'name': '張三', 'score': 85}, {'name': '李四', 'score': 92}, {'name': '王五', 'score': 78} ] def sort_students_by_score(students): for i in range(1, len(students)): key = students[i] j = i - 1 while j >= 0 and students[j]['score'] > key['score']: students[j + 1] = students[j] j -= 1 students[j + 1] = key return students sorted_students = sort_students_by_score(students) for student in sorted_students: print(f"{student['name']}: {student['score']}")

# 插入排序特別適合在線處理（邊接收邊排序） def online_sort_stream(data_stream): sorted_data = [] for value in data_stream: # 使用二分查找找到插入位置 left, right = 0, len(sorted_data) while left < right: mid = (left + right) // 2 if sorted_data[mid] < value: left = mid + 1 else: right = mid sorted_data.insert(left, value) return sorted_data # 處理實時數據流 stream = [42, 28, 65, 15, 78, 35] result = online_sort_stream(stream) print(result) # [15, 28, 35, 42, 65, 78]
